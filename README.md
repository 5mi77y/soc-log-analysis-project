# SOC Log Analysis Project

## Overview
This project demonstrates analysis of SSH logs to detect a brute-force attack, identify a successful compromise, and investigate privilege escalation activity.

## Skills Demonstrated
- Log analysis (SSH authentication logs)
- Threat detection (brute-force attack identification)
- Incident investigation and reporting
- Understanding of privilege escalation (sudo usage)

## Project Files
- `ssh_log_sample-2.log` → Raw log data
- `analysis.md` → Detailed investigation and findings

## Key Findings
- Identified brute-force login attempts from a malicious IP
- Detected successful compromise of a user account
- Observed privilege escalation via sudo commands

## Outcome
This project simulates real-world SOC analysis, demonstrating the ability to detect, investigate, and respond to security incidents.
